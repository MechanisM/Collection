$.Collection (collection, uProp)

$.Collection — JS (JavaScript) framework for working with collections of data (using jQuery).

Glossary:
  • Collection — data object the JS, can be implemented as an array, and as a hash table (you can combine arrays with the hash, for example: [{...},{...},...]);
  • Filter — is a special function, which returns a Boolean value for each "row" of the collection;
  • Parser — is a special function which makes the post—processing of the template;
  • Context — a string which specifies a link to the context of the collection (for example: "Name > 1" indicates the obj.Name[1], where obj is the instance of collection);
  • Template — is a special function, which converts the collection in the text, in accordance with special regulations.
Addition:

The code is documented in accordance with the standard jsDoc.
Specific data types:

  • [Colletion Object] is a reduced form of the [Object] and means an instance of $.Collection;
  • [Colletion] is a reduced form of the [Object|Array] and means an collection of data;
  • [Selector] is a reduced form of the [String], and means the css selector (Sizzle syntax);
  • [Context] is the reduced form of the [String], and means the context of the collection (Nimble syntax);
  • [Template] is a reduced form of the [Function] and means function—template;
  • [Filter] is a reduced form of the [Filter|String] and means the function—filter or string expressions;
  • [Parser] is a reduced form of the [Parser|String] and means function—parser or string expressions;
  • [Plain Object] is a reduced form of the [Object] and means hash table;
  • [jQuery Object] is a reduced form of the [Object] and means an instance of jQuery;
  • [jQuery Deferred] is the reduced form of the [Object] and means an instance of jQuery.Deferred.

For comfortable work it is recommended to use the latest stable version of jQuery.

Enjoy!

$.Collection — JS (JavaScript) фреймворк для работы с коллекциями данных (использует jQuery).

Глоссарий:
  • Коллекция — объект данных JS, может быть представлен, как массив или как хеш–таблица (вы можете комбинировать сущности, например: [{...},{...},...]);
  • Фильтр — специальная функция, которая возвращает логическое значение для каждой «строки» коллекции;
  • Парсер — специальная функция, которая осуществляет постобработку шаблона;
  • Контекст — строка, которая задает ссылку на контекст коллекции (например: "Name > 1" указывает на obj.Name[1], где obj является коллекцией);
  • Шаблон — специальная функция, который преобразует коллекцию в строку, в соответствии со специальными правилами.
Дополнение:

Код оформлен в соответствии со стандартом jsDoc.
Специфичные типы данных:

  • [Colletion Object] является сокращенной формой [Object] и означает экземпляр $.Collection;
  • [Colletion] является сокращенной формой [Object|Array] и означает коллекцию данных;
  • [Selector] является сокращенной формой [String], и означает css селектор (Sizzle синтаксис);
  • [Context] является сокращенной формой [String], и означает контекст коллекции (Nimble синтаксис);
  • [Template] является сокращенной формой [Function] и означает функцию–шаблон;
  • [Filter] является сокращенной формой [Filter|String] и означает функцию–фильтр или строковое выражение;
  • [Parser] является сокращенной формой [Parser|String] и означает функцию–парсер или строковое выражение;
  • [Plain Object] является сокращенной формой [Object] и означает хеш–таблицу;
  • [jQuery Object] является сокращенной формой [Object] и означает экземпляр jQuery;
  • [jQuery Deferred] является сокращенной формой [Object] и означает экземпляр jQuery.Deferred.
Для комфортной работы рекомендуется использовать последнюю стабильную версию jQuery. Наслаждайтесь!
